Weather-App

This is a Weather Application shows real time weather data with the exact speed and humidity of any location. We just need click on search bar and search any city and it will tell the exact current behaviour of the weather. It has the capacity to search any location world wide and show the current weather data. I have used HTML, CSS and JS to make the UI of this website, and i have used OpenWeatherAPI to fetch the current data of any location.
